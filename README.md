# Calculadora de IMC

Projeto de uma Calculadora de IMC (Índice de Massa Corpórea) desenvolvido por mim durante o curso da [B7Web](https://b7web.com.br).

# Tecnologias

- React
- React Hooks
- TypeScript
- CSS Modules
